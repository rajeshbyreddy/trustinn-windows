# TrustInn Desktop - Windows Application

![TrustInn Logo](image.png)

A powerful desktop application for code analysis and verification built with Electron.

## 🚀 Features

- **Code Analysis** - Analyze C, Java, Python, and Solidity code
- **CBMC Integration** - Advanced code verification
- **Real-time Results** - Instant feedback on code analysis
- **Cross-platform** - Works on Windows, macOS, and Linux
- **Secure Authentication** - Built-in user authentication
- **Offline First** - Works without internet connection

## 🛠️ Installation

### Windows Installer
1. Download `TrustInn Setup 1.0.0.exe`
2. Run the installer
3. Follow the setup wizard
4. Launch from Start menu

### Portable Version
1. Download `TrustInn 1.0.0.exe`
2. Run directly (no installation required)
3. Can run from USB drive

## 📋 System Requirements

- **Windows**: Windows 7 or later
- **RAM**: 2GB minimum (4GB recommended)
- **Disk Space**: 200MB

## 🔧 Features & Usage

### Supported Languages
- C (`.c`)
- Java (`.java`)
- Python (`.py`)
- Solidity (`.sol`)

### How to Use
1. Launch TrustInn
2. Login with your NitMiner account
3. Upload or write code
4. Select language and analysis type
5. Review analysis results
